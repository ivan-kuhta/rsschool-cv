Навчальні проекти

  1. Wildlife - виконували звичайну верстку макету
  2. Virtual Piano - писали логіку роботи піаніно на JavaScript з вказаною верстку. Виконували для практики подій.
  3. Photo Filter - писали логіку роботи онлайн фотофільтра на JavaScript з вказаною верстку. Виконували для практики canvas і використання фільтрів в css.
  4. Online Zoo - верстка і написання логіки сайту з макету Figma. Використовував css змінні, для зміни теми сайту. Писали логіку для різноманітних слайдерів.
  5. Match match code - гра написана на чистому JavaScript без верстки елементів, таким чином імітував роботу JavaScript фреймворка (React). В грі використовується IndexDB для реєстрації і логанізації користування. Створював валідацію полів форми. Дані записуються в локальній базі даних. Гра починається через 15 секунд після старту і використовується таймер (практикувалася робота з setTimeout і setInterval). Результати гри записуються в локальну базу і відображаються на сторінці Best Score.
  6. English for Kid - створив повноцінний сайт для вивчення англійської для дітей. В якості Front-end викорстовував React, а Back-end - Express. Що до бази даних використано SQLite. Візуал складається з 2 частин: 1 - основна сторінка, 2 - адмін сторінка. На основній сторінці відбувається робота з API для отримання даних з сервера. На сторінці адміна відбувається додавання, запис і редагування даних. Проект написани на мові TypeScript. Переглянути дані роботи API можна за посиланням. На сервері всі дані зберігаються в SQLite. Для роботи з базою використовував Sequelize. Логанізація відбувається за допомогою JWT.
